SETUP

Make scans real (Free)

  1. Create a free Helius account → get API key.
  2. In Vercel project settings: Environment Variables → add HELIUS_API_KEY.
  3. Redeploy.
  4. Scan a wallet.
This build is Solana-first. DexScreener is used for token market enrichment (free, no key).
